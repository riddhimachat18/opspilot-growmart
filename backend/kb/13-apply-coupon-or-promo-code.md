# Apply a Coupon or Promo Code

**How to apply a coupon at checkout:**
1. Add items to your cart and proceed to checkout
2. Enter your coupon code in the "Apply Coupon" field above the order summary
3. Click "Apply" — the discount will reflect immediately in your order total before payment

**Common reasons a coupon doesn't work:**
- Code has expired or the promotional period has ended
- Order doesn't meet the minimum cart value required for that coupon
- Coupon is restricted to specific product categories not present in your cart
- Coupon is a first-time-user code and has already been used on your account
- Coupon cannot be combined with another ongoing discount or sale price

**Forgot to apply a coupon before payment:** Unfortunately, coupons cannot be applied retroactively to an already-placed order. If the coupon is still valid, you're welcome to use it on your next order. In select cases involving a GrowMart system error (not user error), support may issue the discount difference as store credit — contact support with the coupon code and Order ID to check eligibility.

**Where to find active offers:** Current promotions are listed on the GrowMart homepage banner and in our periodic email/SMS updates for subscribed users.
