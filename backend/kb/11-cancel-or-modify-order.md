# Cancel or Modify an Order

**Cancelling an order:**
- Orders can be cancelled free of charge as long as they haven't been dispatched yet
- Go to "My Orders" → select the order → "Cancel Order," or contact support with your Order ID
- Once an order status shows "Dispatched," it can no longer be cancelled — you'll need to refuse delivery or initiate a return once received
- Refunds for cancelled prepaid orders are processed within 24 hours; see "Refund Policy and Timelines" for how long it takes to reflect in your account

**Modifying an order (address, size/color variant, quantity):**
- Modifications are only possible before an order enters "Processing" status, which typically happens within a few hours of order placement
- Contact support as soon as possible if you need to change anything — once an order is being packed, it cannot be modified, and you'd need to cancel and reorder instead (if not yet dispatched)

**Changing just the delivery address:** see the "Change Shipping Address" article for a faster path if that's the only change needed.

If you're unsure of your order's current status, check "My Orders" — the cancellation/modification option will only appear if the order is still eligible.
