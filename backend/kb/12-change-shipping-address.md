# Change Shipping Address

**Before an order is placed:** You can add, edit, or set a default address anytime under "My Account" → "Saved Addresses."

**After an order is placed but not yet dispatched:**
- Contact support with your Order ID and the corrected address as soon as possible
- Address changes are only possible while the order is in "Order Confirmed" or early "Processing" status — once packed for dispatch, the address is locked in the courier partner's system and cannot be changed
- There is no charge for an address correction made in time

**After an order is dispatched:**
- The delivery address generally cannot be changed once handed to the courier partner
- In some cases, the courier partner may allow a "delivery instructions" update (e.g., alternate contact number, landmark) but not a full address change — contact support and we'll attempt this with the courier, though it isn't guaranteed
- If delivery fails due to an incorrect address, the package is typically returned to our warehouse, and you can either request redelivery to the correct address (redelivery shipping fee may apply) or a refund

To avoid address issues, double-check your address and pin code carefully at checkout, especially apartment/floor numbers and landmark details.
